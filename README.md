# Sistema de Filtrado Inteligente para Senior Co-housing (GII + ADE)

Diseño y desarrollo de un prototipo de filtrado inteligente para el emparejamiento de convivencia senior (Senior Co-Housing) mediante sistemas basados en conocimiento.


## Descripción del Proyecto
Contenido del desarrollo  del TFG sinérgico del Doble Grado en ADE e Ingeniería Informática. El objetivo es automatizar la toma de decisiones en el emparejamiento de inquilinos en modelos de Senior Co-housing, garantizando una convivencia óptima.


##  Tecnologías Utilizadas
- **Protégé:** Para el diseño y validación de la ontología.
- **Python / Owlready2:** Para la integración del motor de reglas y la manipulación del modelo OWL.
- **W3C Standards:** Uso de estándares OWL, RDF y SPARQL.

##  Estructura del Repositorio
- `/ontology`: Archivos fuente de la ontología (.owl).
- `/src`: Código fuente del motor de reglas y lógica del prototipo.
- `/docs`: Documentación técnica, requisitos y borradores de la memoria.
- `/data`: Datos de prueba (perfiles anonimizados para testing).
